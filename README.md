# Glacier::CLI

Basic CLI for AWS Glacier to back up files.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'glacier-cli'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install glacier-cli
